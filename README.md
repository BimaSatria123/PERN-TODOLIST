# PERN Todo List

Aplikasi Todo List menggunakan:

- PostgreSQL
- Express.js
- React.js
- Node.js

## Struktur Project

```text
PERN-TODOLIST
├── Backend
└── Frontend
```

## Menjalankan Backend

```bash
cd Backend
npm install
npm start
```

## Menjalankan Frontend

```bash
cd Frontend
npm install
npm run dev
```
